Proceso de potabilización


Potabilización

La potabilización es un proceso que sirve para potabilizar el agua cruda, agua que no ha sido sometida a ningun proceso, y que muchas veces excede algun parametro de los establecidos por la normativa.

ver normativa mexicana

La potabilización puede ser de diferentes tipos por lo que varían los procesos y métodos que la conforman, a continuación se muestran algunos de los más comunes :

1. Métodos Físicos

  • Filtración: Utiliza filtros (de arena, grava, carbón activado) para eliminar sólidos suspendidos e impurezas.
  • Sedimentación: Permite que las partículas más pesadas se asienten en el fondo de un tanque.
  • Destilación: Hierve el agua para que se evapore y luego condensa el vapor, dejando atrás los contaminantes.
  • Evaporación Solar: Utiliza calor solar para evaporar el agua y recoger el vapor condensado.
  • Aireación: Introduce aire al agua para eliminar gases disueltos.

2. Métodos Químicos

  • Cloración: Adición de cloro para destruir bacterias y virus.
  • Uso de Yodo o Pastillas Purificadoras: Se añaden compuestos químicos para desinfectar el agua.
  • Ozono: Utiliza ozono como desinfectante potente sin dejar residuos químicos.
  • Coagulación/Floculación: Se añaden coagulantes para agrupar partículas finas en flóculos.
  • Ajuste de pH: Se ajusta el pH para precipitar metales y mejorar la calidad del agua.

3. Métodos Biológicos

  • Filtros de Arena Lenta: Utilizan microorganismos para eliminar bacterias y materia orgánica.
  • Tratamiento con Plantas Acuáticas: Algunas plantas pueden absorber contaminantes y purificar el agua.
  • Biopelículas: Aprovechan comunidades microbianas para degradar contaminantes orgánicos.

4. Métodos de Desinfección Avanzada

  • Luz Ultravioleta (UV): Utiliza radiación UV para inactivar bacterias, virus y protozoos.
  • Electrólisis: Utiliza corriente eléctrica para generar agentes desinfectantes en el agua.
  • Nanofiltración y Ósmosis Inversa: Utilizan membranas para eliminar sales, virus y otras impurezas.

5. Métodos Caseros

  • Hervido: Hervir el agua durante 1-3 minutos para eliminar patógenos.
  • Filtración con Filtros Portátiles: Uso de filtros de carbón activado o cerámica para eliminar sedimentos y microorganismos.
  • Uso de Carbón Activado: Filtra el agua para eliminar olores, sabores y algunos compuestos químicos.
